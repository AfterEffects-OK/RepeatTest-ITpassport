# Quiz Busters - Tactical Cockpit

「Quiz Busters」は、サイバーパンクな戦闘機のコックピットを舞台にした、ブラウザベースのシューティングクイズゲームです。

プレイヤーはパイロットとなり、次々と飛来する「正解の選択肢」を撃ち落としてスコアを稼ぎます。問題とスコアの記録にはバックエンドとしてGoogleスプレッドシートを利用しており、問題の追加やランキングの管理が容易に行えます。

## 主な機能

- **ダイナミックなクイズ出題**: Googleスプレッドシートから問題を動的に読み込みます。
- **スコア記録とランキング**: ゲームの結果はスプレッドシートに自動で記録され、プレイヤーごとの最高スコアに基づいたランキングを閲覧できます。
- **カテゴリ機能**: スプレッドシートで設定したカテゴリごとに問題を絞り込んでプレイできます。
- **リザルトとレビュー**: ゲームオーバー時にスコアや命中率が表示され、間違えた問題の解説を確認できます。
- **ボーナスシステム**: 20回正解するごとにボーナススコアが、4000ポイント獲得するごとにライフが回復します。
- **プロシージャルサウンド**: 発射音やヒット音などの効果音は、Web Audio APIを使用して動的に生成されます。

## 技術スタック

- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
  - 描画: Canvas API
  - 音響: Web Audio API
- **バックエンド**: Google Apps Script (GAS)
- **データベース**: Google Sheets

## セットアップ方法

このゲームを動作させるには、バックエンド（GoogleスプレッドシートとGAS）とフロントエンド（HTML/JS/CSS）の両方を設定する必要があります。

### 1. バックエンドのセットアップ (Google Sheets & Apps Script)

1.  **Googleスプレッドシートの作成**:
    - 新しいGoogleスプレッドシートを作成します。

2.  **`scores` シートの作成**:
    - 最初のシートの名前を `scores` に変更します。
    - 1行目に、以下のヘッダーをA列から順に設定します。
      > `IDスコア`, `タイムスタンプ`, `ユーザーネーム`, `メールアドレス`, `スコア`, `プレイ時間`, `命中率`, `命中数`

3.  **`問題集` シートの作成**:
    - 新しいシートを追加し、名前を `問題集` に変更します。
    - 1行目に、以下のヘッダーをA列から順に設定します。（AppSheet連携を想定したカラムも含まれています）
      > `ID問題集`, `問題文`, `正解`, `不正解1`, `不正解2`, `不正解3`, `不正解4`, `不正解5`, `不正解6`, `不正解7`, `不正解8`, `不正解9`, `不正解10`, `不正解11`, `不正解12`, `不正解13`, `不正解14`, `不正解15`, `解説文`, `ファイルリンク`, `ファイルパス`, `作成日時`, `編集日時`, `作成者Email`, `編集者Email`, `フラグ`, `メモ`, `カテゴリ`, `表示・非表示フラグ`
    - **AC列 (`表示・非表示フラグ`)** にはチェックボックスを設定してください（`挿入` > `チェックボックス`）。チェックが **OFF** の問題が出題対象となります。

4.  **Google Apps Script (GAS) の設定**:
    - スプレッドシートの `拡張機能` > `Apps Script` を開きます。
    - `コード.gs` ファイルの内容をエディタに貼り付けて、プロジェクトを保存します。

5.  **ウェブアプリとしてデプロイ**:
    - Apps Scriptエディタの右上にある `デプロイ` > `新しいデプロイ` をクリックします。
    - `種類の選択`（歯車アイコン）をクリックし、`ウェブアプリ` を選択します。
    - **説明**: （任意）`QuizBusters Backend` など
    - **次のユーザーとして実行**: `自分`
    - **アクセスできるユーザー**: `全員`
    - `デプロイ` ボタンをクリックします。
    - 権限の承認を求められたら、許可してください。
    - 表示された**ウェブアプリURL**をコピーします。これは次のステップで使用します。

### 2. フロントエンドのセットアップ

1.  **ファイルの配置**:
    - `index.html`, `style.css`, `script.js` の3つのファイルを同じフォルダに配置します。

2.  **バックエンドURLの設定**:
    - `script.js` ファイルを開き、以下の部分を見つけます。
      ```javascript
      const GAS_WEB_APP_URL = 'https://script.google.com/macros/s/AKfy.../exec';
      ```
    - `''` の中に、先ほどコピーした**ウェブアプリURL**を貼り付けます。

3.  **ゲームの起動**:
    - `index.html` ファイルをウェブブラウザで開きます。

## 遊び方

1.  **パイロット情報の入力**:
    - `PILOT NAME` と `COCKPIT ACCESS EMAIL` を入力します。これらの情報はランキングの個人識別に使われます。

2.  **カテゴリの選択**:
    - プレイしたいクイズのカテゴリを選択します。「全カテゴリ」を選ぶと、すべての問題から出題されます。

3.  **ゲーム開始**:
    - `INITIALIZE SYSTEM` ボタンをクリックしてゲームを開始します。

4.  **操作**:
    - **照準**: マウスカーソルを動かして照準を合わせます。
    - **射撃**: マウスの左クリックでミサイルを発射します。

5.  **ルール**:
    - 画面中央に表示される問題文を読み、飛来する選択肢の中から**正解のターゲット**を撃ち抜いてください。
    - 不正解のターゲットを撃つか、正解のターゲットを撃ち逃して画面外に消えるとライフが減少します。
    - ライフが0になるとゲームオーバーです。

6.  **ボーナス**:
    - **20 HITS BONUS**: 正解数（HITS）が20に達するごとに、ボーナススコアが加算されます。
    - **LIFE UP**: スコアが4000ポイントに達するごとに、ライフが1つ回復します。
